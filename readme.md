#kc.io

kc.io is a simple jobs app written in ruby

It is a work in progress.  Hack it up all you want. 

Tools used:

- ruby
- sinatra
- twitter bootstrap
- font awesome
- datamapper
- heroku
- postgresql
- haml

Key gems:

- sinatra-authentication
